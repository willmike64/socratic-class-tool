# Socratic Opportunity Identification & Evaluation (Streamlit)

## Run locally
1. Install dependencies:
   - `pip install streamlit`
2. Start the app:
   - `streamlit run socratic_opportunity_streamlit_app.py`

## What it includes
- Socratic discussion flow (Student + Instructor mode)
- Student worksheet generator + download
- Instructor guide + answer key downloads
